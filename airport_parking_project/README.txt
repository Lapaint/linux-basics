# 전국 공항 주차장 실시간 정보 시스템 - 리눅스 기말과제

## 👥 팀명: 7조
- 김민우
- 이준엽
- 임재영
- 정민수

---

## 📌 프로젝트 개요

공공데이터포털의 “전국 공항 주차장 실시간 정보 API”를 활용하여,
공항별 주차장의 실시간 남은 자리와 혼잡도를 시각화하는 자동화된 웹 페이지를 구축했습니다.

해당 시스템은 `Docker`, `cron`, `bash script`, `nginx` 등을 활용하여 주기적으로 정보를 갱신하고 정적 웹페이지를 제공합니다.

---

## 🛠 기술 구성

- **Docker Compose**: 데이터 수집 및 웹 서버를 각각의 컨테이너로 구분하여 관리
- **Shell Script**: XML 데이터 파싱 → HTML 페이지 자동 생성
- **crontab**: 5초 주기로 스크립트 실행하여 최신 정보 갱신
- **nginx**: 정적 웹페이지 호스팅
- **공공데이터포털 API**: 전국 공항 주차장 실시간 정보 제공

---

## 🧩 주요 기능

1. **공항 주차 정보 테이블**  
   - 공항명, 주차장명, 남은 자리 수, 혼잡도 표시  
   - 혼잡도: 여유 / 보통 / 혼잡 (색상 표시)

2. **검색 기능**
   - 입력창에 공항명 입력 시 해당 공항만 필터링
   - 검색 결과가 없으면 경고창 출력
   - Enter 키 or 버튼 클릭으로 검색 가능

3. **정렬 기능**
   - 혼잡도: 여유 < 보통 < 혼잡 순서로 정렬 (토글 방식)
   - 남은 자리: 많음 → 적음, 다시 클릭 시 적음 → 많음
   - 공항명 / 주차장명 알파벳순 정렬도 가능

4. **Google 지도 연동**
   - 주차장명을 클릭하면 해당 주차장의 Google 지도 검색 링크로 이동

5. **자동 갱신**
   - 페이지는 5초마다 새로고침
   - **현재 정렬 상태 및 검색 결과 유지**

6. **마지막 갱신 시간 표시**
   - 상단에 마지막 데이터 갱신 시간 표시 (ex. 마지막 갱신: 2025-06-04 14:03:00)
